Slack DM Sender

A Slack bot that sends direct messages as you — pick recipients with filters, fire now or schedule for later, all rate-limited so you never trip Slack's caps.

Watch the demo View on GitHub

Slack API · OAuth · Node.js · Redis & BullMQ

What it does

💬

DM as you

Receives the message you want to send, then delivers it as a direct message from your own Slack identity — not a faceless bot.

🎯

Filter & select

Choose the exact set of recipients with filter options, then trigger the send immediately or schedule it for a time you pick.

🚦

Rate-limited by design

A Redis-backed BullMQ queue paces every message so you stay comfortably inside Slack's API limits, even for large recipient lists.

🔐

OAuth & access control

Workspace OAuth handles authentication, and an access-control layer with platform-based handling gates who can send what.
